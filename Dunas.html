Rediseño web del

El desafío es rediseñar la página web del Hotel Dunas de Oyambre. Para hacer frente, hemos optado por realizar un Análisis Heurístico, a través del cual buscamos identificar problemas de usabilidad para optimizar el proceso de rediseño. Esperamos que este enfoque nos permita ahorrar tiempo y recursos a largo plazo al abordar los problemas y garantizar una experiencia de usuario mejorada.
