DISEÑO WEB

roll Diseñador web y gráfico

herramientas HTML | CSS | JS

habilidades Web Design | UI Design

Para demostrar mis habilidades en diseño web y programación, decidí desarrollar mi propio currículum vitae desde cero, utilizando Visual Studio Code, he empleado HTML, CSS y JavaScript para crear una experiencia interactiva y visualmente atractiva. Este proyecto no solo refleja mis competencias técnicas, sino también mi capacidad para abordar desafíos creativos.

sitema de diseño

Utilicé HTML para estructurar el contenido de manera semántica y CSS para estilizar y dar formato a los elementos e incorporé JavaScript para agregar interactividad, como animaciones y funcionalidades específicas.

La web exhibe una selección de trabajos realizados en diversos campos del diseño, organizados en tres secciones principales: página principal, trabajos y sobre mí. Cada sección está diseñada para ofrecer una experiencia visual y de navegación fluida y coherente. Asimismo, la página es completamente responsive, adaptándose automáticamente al tamaño de la pantalla, ya sea en versión desktop, iPad o móvil y podrán apreciar variaciones específicas en el diseño entre cada versión, asegurando una experiencia adaptada en todos los contextos de uso.

Para obtener más detalles sobre este proyecto, los invito a revisar esta misma página.
